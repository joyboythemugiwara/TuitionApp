# TuitionApp

A comprehensive full-stack platform for managing tuition centers, students, batches, and fee tracking.

## Features
- **Student Management:** Register, update, and manage student profiles.
- **Batch Management:** Organize students into different batches.
- **Fee Tracking:** Easily track fee payments and generate records.
- **Dashboard Overview:** Get insights into your tuition center's overall performance.

## Project Structure
This is a full-stack application structured as a monorepo:
- `/frontend` - The frontend application (built with Next.js/React)
- `/backend` - The backend API service (Node.js/Express)

## Getting Started

### Prerequisites
- Node.js (v18+)
- bun or npm

### Setup

1. **Clone the repository:**
   ```bash
   git clone git@github.com:joyboythemugiwara/TuitionApp.git
   ```

2. **Frontend Setup:**
   ```bash
   cd frontend
   bun install
   bun run dev
   ```

3. **Backend Setup:**
   ```bash
   cd backend
   bun install
   bun run dev
   ```
